# Fonte de Tensão Ajustável
Desenvolvimento de uma fonte de tensão ajustável de 3 V a 12 V, com capacidade de fornecer até 100 mA de corrente à carga, alimentada pela rede elétrica de 127 V.

## Descrição:
Trabalho 1 da Disciplina de Eletrônica para Computação.
